# Increment 1: Minimal server

## Goal
Run a local server with:
- GET /health
- POST /extract (placeholder)

## Why
Learn backend fundamentals before adding OCR or LLMs.

## What we are building
- simple Express server
- no business logic

## Eval
- GET /health returns { ok: true }
- POST /extract returns 501

## Ship it
Server runs locally and endpoints behave correctly.

## Do not add yet

- file upload
- LLMs
- frontend
