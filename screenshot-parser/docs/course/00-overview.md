# Overview

## Goal
Build an app that extracts:
- topics
- URLs
- people
- raw text
- stitched document from screenshots

## Development ladder
1. Minimal server
2. File upload
3. OCR
4. Structured extraction
5. Traces + evals
6. Multi-image
7. Stitching
8. UI

## Rule
Do not skip steps. Each increment must run and be verifiable.
